# nanorc
Configuration file for the nano text editor. Besides some simple tweaks, its main goal is to bring key-bindings closer to Emacs. Syntax highlighting for the Fortran programming language is also added.

## How to use

Type the following command on your terminal:

>
> wget -O ~/.nanorc https://raw.githubusercontent.com/inigogonzalezdearrieta/nanorc/refs/heads/main/.nanorc 
>
